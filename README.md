# Threads Autopilot Assets Template

Threads自動投稿環境で使用する、公開画像専用のテンプレートリポジトリです。

## 用途

このリポジトリには、Threads投稿時に外部から参照するWebP画像だけを保存します。

```text
assets/
└─ webp/
   └─ <image_key>.webp
```

## 公開に関する注意

このリポジトリはPublicです。追加した画像は、Threadsへ投稿する前でもインターネット上から閲覧できる状態になります。

次のものは保存しません。

- 投稿本文、投稿予定、投稿履歴
- 顧客情報、個人情報、機密情報
- パスワード、アクセストークン、GitHubトークン、Secrets
- 利用権利を確認できない画像
- 公開前に第三者へ見られて困る画像

## 運用

- 顧客ごとに、このテンプレートから専用のPublicリポジトリを作成します。
- Private側の自動投稿リポジトリが、画像をWebPへ変換して`assets/webp/`へ配置します。
- 画像URLが作成されても、投稿状態は自動で`ready`へ変更しません。
- 投稿前に、本文・日時・画像・投稿先を確認します。

通常の利用者へは、Goroが初期設定を完了した後、完成したリポジトリURLだけを案内します。
